\doxysubsection{Detailed Description}
Manages the SQLite database connection and provides methods for user, deck, card, and study session management. 

The \mbox{\hyperlink{classDatabase}{Database}} class is responsible for connecting to an SQLite database and providing various methods to manage users, decks, cards, and study sessions. It also includes methods for spaced repetition and card progress updates.

\begin{DoxyNote}{Note}
This class requires the SQLite3 library. 
\end{DoxyNote}
