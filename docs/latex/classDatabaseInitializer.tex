\doxysubsection{Detailed Description}
A utility class for initializing and checking the state of a database. 

This class provides static methods to initialize a database and to check if a database has already been initialized.

Initializes the database by creating necessary tables if they do not exist. 

This function creates the following tables\+:
\begin{DoxyItemize}
\item users\+: Stores user information with columns for id, username, and password.
\item decks\+: Stores deck information with columns for id, user\+\_\+id, and name. It references the users table and deletes decks if the associated user is deleted.
\item cards\+: Stores card information with columns for id, deck\+\_\+id, question, answer, interval, ease\+Factor, repetitions, lapses, and due\+Date. It references the decks table and deletes cards if the associated deck is deleted.
\end{DoxyItemize}
