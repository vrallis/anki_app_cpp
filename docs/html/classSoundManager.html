Detailed Description

Manages the loading and playing of sound effects.

The SoundManager class provides functionality to load sound effects from files and play them using their associated names.

SoundManager::SoundManager ( )

Constructs a new SoundManager object and initializes the SDL audio subsystem.

This constructor initializes the SDL audio subsystem and the SDL_mixer library. If SDL or SDL_mixer fails to initialize, an error message is printed to the standard error stream.

Note
SDL_Init(SDL_INIT_AUDIO) initializes the SDL audio subsystem.
Mix_OpenAudio(44100, MIX_DEFAULT_FORMAT, 2, 2048) initializes the SDL_mixer library with a frequency of 44100 Hz, the default format, 2 channels (stereo), and a chunk size of 2048 bytes.

◆ ~SoundManager()

SoundManager::~SoundManager ( )

Destructor for the SoundManager class.

This destructor is responsible for cleaning up all the resources associated with the SoundManager. It iterates through the stored sounds, frees each sound chunk, clears the sounds container, and shuts down the audio and SDL subsystems.

Loads a sound file and stores it in the sound manager.

This function attempts to load a sound file from the specified path and associates it with a given sound name. If the file does not exist or fails to load, an error message is printed to the standard error stream.

void SoundManager::playSound ( const std::string &  soundName)

Plays the specified sound if it exists in the sound manager.

This function checks if the sound with the given name exists in the sound manager's collection of sounds. If the sound is found, it plays the sound on any available channel. If the sound is not found, it outputs an error message to the standard error stream.
