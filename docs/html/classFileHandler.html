std::vector< std::pair< std::string, std::string > > FileHandler::importCards ( const std::string &  filePath)
static

Imports cards from a specified file.

This function reads a file containing card data, where each line represents a card with a question and an answer separated by a comma. It parses the file and stores the cards in a vector of pairs, where each pair consists of a question and an answer.

Parameters
filePathThe path to the file containing the card data.
Returns
A vector of pairs, where each pair contains a question and an answer. If the file cannot be opened or if there are lines with invalid format, the function will return an empty vector or a partially filled vector.
