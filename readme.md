# Anki App

## Installation Instructions

### English

1. **Clone the repository:**
    ```bash
    git clone https://github.com/vrallis/anki_app_cpp
    cd anki_app_cpp
    ```

2. **Build the application:**
    ```bash
    mkdir build
    cd build
    cmake ..
    make
    ```

3. **Run the application:**
    ```bash
    ./anki_app_cpp
    ```
